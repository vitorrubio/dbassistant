# Critério de Aceite

## Teste de Integração
Crie um integration test que execute o método QueryAsync do AssistantController com um teste para cada uma das perguntas em linguagem natural abaixo. O critério de aceite é todas essas perguntas serem respondidas com uma query SQL válida e os resultados corretos. Para isso o teste deve comparar os resultados da api com os resultados uma consulta direta ao banco.

1. Quais são os produtos mais populares entre os clientes corporativos?
2. Quais são os produtos mais vendidos em termos de quantidade?
3. Qual é o volume de vendas por cidade?
4. Quais são os clientes que mais compraram?
5. Quais são os produtos mais caros da loja?
6. Quais são os fornecedores mais frequentes nos pedidos?
7. Quais os melhores vendedores?
8. Qual é o valor total de todas as vendas realizadas por ano?
9. Qual é o valor total de vendas por categoria de produto?
10. Qual o ticket médio por compra?
11. Qual é o pior vendedor em quantidade de vendas?
12. Qual é o pior vendedor em valor total de vendas no último mês?
