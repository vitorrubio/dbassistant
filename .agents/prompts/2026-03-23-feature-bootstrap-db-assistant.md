# Internal Prompt Notes

## Objective
Create the first project scaffold for DB Assistant following Clean Architecture, DDD, interface-driven design, thin controllers, and test coverage requirements.

## Constraints
- No data access code in controllers.
- One class per file.
- Use MySQL.
- Only placeholders in `.env`.
- Keep `.agents` updated to avoid re-reading completed work.
